# CPO 技术原理与产业趋势

> **定位**：CPO 板块的技术基础研究。理解产业链价值重构的逻辑起点。

---

## 一、为什么需要 CPO？—— 算力爆炸下的互联瓶颈

### 1.1 AI 算力的三重物理瓶颈

AI 大模型训练和推理要求 GPU 之间以 **Tb/s 级别**带宽互联，传统可插拔光模块面临三重物理瓶颈：

#### ① 功耗瓶颈（最核心）
- 传统方案电信号要通过 **PCB 铜线 + SerDes（串行/解串器）** 长距离传输
- SerDes 能耗随速率 **指数级**上升：224G PAM4 单通道功耗已达 5W+
- 交换芯片 I/O 功耗占比超过 **50%**，800G 后不可持续

#### ② 带宽密度瓶颈
- 交换芯片单颗带宽容量演进：12.8T → 25.6T → 51.2T → 102.4T
- **面板 I/O 光模块数量**已触及物理上限（32 个 800G 光口已是极限）
- 1.6T 以上速率，铜线材料本身的损耗和串扰无法承受

#### ③ 空间/散热瓶颈
- 机柜密度持续提升，光模块散热空间压缩
- GB200 NVL72 单机柜功耗达 120KW，传统布线接近极限

### 1.2 解决思路的演进

```
可插拔光模块（Pluggable）
    ↓ 光模块下沉到 PCB
NPO（Near-Packaged Optics）近封装光学
    ↓ 进一步贴近 ASIC
CPO（Co-Packaged Optics）共封装光学
    ↓ 光引擎与 ASIC 同一基板
最终形态：光 I/O（Optical I/O，Ayar Labs 方向）
```

## 二、CPO 核心技术架构

### 2.1 CPO 交换机结构示意

```
┌──────────────────────────────────────────┐
│  CPO 交换机 ASIC 封装基板                  │
│  ┌────────────┐   ┌──────────────┐       │
│  │            │←──│  光引擎 OE   │── 光纤 │
│  │  交换 ASIC │   │ (PIC + EIC) │── 光纤 │
│  │  51.2T     │←──│              │── 光纤 │
│  │            │   └──────────────┘       │
│  └────────────┘           ↑              │
│         ↑                  │              │
│         └─ 电信号路径 <1cm │              │
│                            │              │
│         外置光源 ELS ──────┘              │
└──────────────────────────────────────────┘
```

### 2.2 核心组成要素

#### ① 光引擎（Optical Engine, OE）—— CPO 心脏
由 **光子集成电路 PIC** + **电子集成电路 EIC** 组成：
- **PIC（Photonic IC）**：硅光调制器、光电探测器、波导、耦合器
- **EIC（Electronic IC）**：TIA 跨阻放大器、Driver 驱动器
- 两者通过 **2.5D/3D 先进封装**键合在一起

**主流方案**：硅光（Silicon Photonics）是 CPO 光引擎的主流技术路径

#### ② 外置光源 ELS（External Laser Source）
CPO 的激光器**不能**集成在 ASIC 封装内部（温度太高，寿命短），必须外置：
- **主流方案**：CW-DFB 激光器（连续波分布式反馈激光器）
- **大功率需求**：单颗 CW 光源输出功率要求从 10mW → 70mW+
- **国产核心标的**：**源杰科技、仕佳光子**

#### ③ FAU（Fiber Array Unit）光纤阵列单元
将数十/上百根光纤按微米级精度排列，耦合到硅光芯片：
- **精度要求**：125μm/250μm 间距，亚微米级对位
- **国产核心标的**：**天孚通信、太辰光、仕佳光子**

#### ④ MT-Ferrule / MPO 连接器
高密度光纤连接器，单接口支持 12/16/24 芯光纤：
- **国产核心标的**：**致尚科技（福可喜玛 MT 插芯）、太辰光**

#### ⑤ 先进封装（Advanced Packaging）
- **台积电 COUPE**（Compact Universal Photonic Engine）：CPO 旗舰工艺
- **Intel Co-EMIB**：Intel 方案
- **2.5D 中介层（Interposer）+ 3D 混合键合**

## 三、技术路线之争：CPO vs. LPO vs. NPO

### 3.1 三种方案对比

| 方案 | 全称 | 原理 | 功耗节省 | 成熟度 | 目标场景 |
|---|---|---|---|---|---|
| **LPO** | Linear Pluggable Optics | 去掉 DSP，线性直驱 | ~30% | 2024 量产 | 机柜内、短距 |
| **NPO** | Near-Packaged Optics | 光模块贴近 ASIC | ~40% | 2025-2026 | 过渡方案 |
| **CPO** | Co-Packaged Optics | 光引擎与 ASIC 共封装 | **40-50%** | 2026-2028 | 机架间、集群级 |

### 3.2 为何市场最终会走向 CPO？

- **LPO** 只能在短距场景使用（<100m），且带宽上限受限
- **NPO** 只是过渡方案，不解决根本的封装级互连问题
- **CPO** 是物理极限下的终极解决方案，尤其在 1.6T 以上速率

**共识结论**：**LPO / NPO / CPO 在不同场景并存，但长期 CPO 在超高带宽场景主导。**

## 五、价值链重构：钱流向哪里？

### 5.1 传统可插拔架构价值分布

```
交换芯片 ~15%  |  DSP ~10%  |  光模块 ~60%  |  光纤连接 ~15%
                              ↑
                            价值核心
```

### 5.2 CPO 架构价值重分布

```
交换芯片 ~20%  |  光引擎/硅光 ~30%  |  外置光源 ~15%  |  FAU/连接 ~15%  |  封装 ~20%
                        ↑                  ↑                    ↑                ↑
                    价值跃升            新增刚需            国产替代           台积电/封测
```

### 5.3 投资启示

1. **硅光 + 先进封装**成为新的价值高地
2. **上游 CW 光源与精密光学**出现"从 0 到 1"的增量需求
3. **传统光模块厂商**必须向光引擎/封装延伸，否则价值被侵蚀
4. **设备与材料**先于模块兑现业绩（罗博特科、炬光逻辑）

## 六、风险与不确定性

### 6.1 技术风险
- 硅光 CPO 封装良率（当前约 70-80%，目标 95%+）
- 光引擎与 ASIC 热应力导致的可靠性问题
- CW 光源寿命（目标 10 万小时）

### 6.2 产业风险
- 大客户技术路线切换（英伟达、博通之间）
- LPO 在中短距场景对 CPO 的分流
- 800G 光模块价格战传导至 CPO 定价

### 6.3 估值风险
- 2026-04 龙头标的 PE-TTM 普遍在 **70-100 倍**区间
- 业绩兑现节奏若低于预期，回撤压力较大
