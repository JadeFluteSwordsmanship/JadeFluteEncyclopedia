# 光引擎与光模块环节（中游）

> **环节定位**：CPO 产业链的**价值核心**与**业绩兑现核心**。传统光模块龙头通过光引擎能力向 CPO 升级。
> 
> **核心逻辑**：短期「易中天」三剑客靠 800G/1.6T 可插拔业绩兑现；中长期谁能拿下英伟达/博通 CPO 光引擎供应，谁就赢得第二增长曲线。

---

## 一、环节地图

```
ASIC 交换芯片
    ↓ SerDes 电接口
┌───────────────────────────┐
│  光引擎 (Optical Engine) │ ← 本环节核心
│  PIC (硅光) + EIC (电芯片) │
└───────────────────────────┘
    ↓
外置光源 ELS → 耦合 → FAU → 光纤
```

**价值排序（CPO 架构下）**：光引擎 > 整体光模块 > 外置光源 > FAU > 连接器

---

## 二、「易中天」三剑客 —— 光模块全球龙头

### 1. 中际旭创（300308.SZ）—— 全球光模块市占率第一

| 项目 | 详情 |
|---|---|
| **行业地位** | 2025 年全球光模块市占率 **23.4%**，连续 3 年全球出货量第一 |
| **核心客户** | 谷歌（含 TPU v7 订单）、亚马逊、Meta、英伟达供应链 |
| **产品矩阵** | 400G / 800G / 1.6T 可插拔光模块，硅光方案，**适配 CPO 的光引擎** |
| **CPO 布局** | 自研硅光光引擎，通过**光引擎封装**切入 CPO 核心价值链 |
| **2025 业绩** | 营收 **382.4 亿**（+60.25%）； 归母净利 **107.97 亿**（+108.78%）；毛利率 **42.61%** |
| **2025 Q4** | 营收 132.4 亿（+102% yoy），净利 36.65 亿（+158% yoy），**毛利率 44.48%** |
| **增长驱动** | 800G 放量 + 1.6T Q3 起 Q4 爬坡 |
| **估值** | PE-TTM ~76 倍（2026-04） |
| **2025 股价涨幅** | **+397.51%** |
| **券商预期** | 高盛预计 2025-2028 净利 CAGR **53%**；2026 年全球 800G+ 光模块销量 +101% |

**核心看点**：
- ✅ 规模与客户结构最优，毛利率持续向上
- ✅ 1.6T 快速起量验证产品迭代能力
- ⚠️ 3.2T 进度略慢于同行（2026 H2 进客户测试）
- ⚠️ 存货 126.81 亿（+80%），存在跌价风险

**跟踪指标**：800G/1.6T 出货占比、毛利率、谷歌 TPU v7 + 英伟达 GB300 订单能见度

### 2. 新易盛（300502.SZ）—— 北美云厂深度绑定

| 项目 | 详情 |
|---|---|
| **行业地位** | 全球光模块第二梯队龙头，北美云厂核心供应商 |
| **核心客户** | **深度绑定博通**，北美 CSP（Amazon、Meta） |
| **产品矩阵** | 400G/800G/1.6T 光模块，**1.6T 硅光方案样品**已推出 |
| **CPO 布局** | 硅光方案推进，跟随博通 CPO 交换机生态 |
| **2025 Q3** | 营收 **60.68 亿**（+152.53% yoy），业绩爆发 |
| **2025 业绩预告** | 净利润大幅增长，具体数据待正式年报 |
| **估值** | PE-TTM 70-80 倍区间 |
| **核心弹性** | 博通 CPO 订单 + 北美云厂 1.6T 放量 |

**核心看点**：
- ✅ 绑定博通 Bailly 51.2T CPO 交换机，弹性空间大
- ✅ 北美客户结构（毛利率高）
- ⚠️ 客户集中度较高（博通、AWS）

**跟踪指标**：博通 CPO 订单进展、1.6T 硅光方案量产时间、博通季报

### 3. 天孚通信（300394.SZ）—— CPO 纯度最高的弹性龙头

| 项目 | 详情 |
|---|---|
| **公司定位** | 光器件平台型企业，从无源器件向**光引擎、有源器件**延伸 |
| **CPO 核心价值** | **英伟达 CPO 光引擎 + FAU 核心主力供应商** |
| **产品矩阵** | 高速光引擎、FAU 光纤阵列、光学元件、隔离器、PLC 等全品类 |
| **战略客户** | **英伟达（CPO 核心供应链）**、Coherent、中际旭创、新易盛 |
| **CPO 单机价值量** | 英伟达 Quantum-X CPO 交换机**单机配套价值量显著高于传统模块** |
| **业绩增长** | 核心得益于 CPO + 光纤相关产品订单爆发 |

**核心看点**：
- ✅ **CPO 纯度最高**，受益弹性最大
- ✅ 英伟达独家/核心供应地位
- ✅ 从器件到光引擎平台化延伸
- ⚠️ 2025 业绩预告一度不及预期（2026-01）引发市场担忧，但 CPO 长逻辑未变

**跟踪指标**：英伟达 Quantum-X / Spectrum-X 出货量、FAU 单机价值量、光引擎毛利率

## 三、二线光模块 & 光引擎标的

### 4. 华工科技（000988.SZ）
- 激光+光通信双主业，光模块品类齐全
- CPO 光引擎在研，国资背景有战略价值
- 估值相对合理，适合稳健配置

### 5. 剑桥科技（603083.SH）
- CPO 光模块 + 光引擎研发方向
- 深耕 800G/1.6T，与海外客户有合作
- 弹性大，但业绩波动性较高

### 6. 光迅科技（002281.SZ）
- 央企（中国信科旗下）光电子器件龙头
- 自研硅光芯片、EML 芯片，**薄膜铌酸锂**布局领先
- **国产自主可控**主线最强标的，估值相对温和
- 既是中游光模块厂，也是上游光芯片厂，全产业链一体化

### 7. 联特科技（301205.SZ）
- 高速光模块厂商，产品线完善
- 规模较小，弹性较大

### 8. 华是科技 / 博创科技（300548.SZ）
- 光模块二线厂商，硅光方向有布局

## 四、环节内部分化逻辑

### 4.1 传统光模块厂商的分层

| 层次 | 代表 | 逻辑 |
|---|---|---|
| **T0 全球龙头** | 中际旭创、新易盛 | 规模+客户+技术三重护城河 |
| **T0.5 平台化** | 天孚通信 | 从器件向引擎延伸，CPO 纯度最高 |
| **T1 国资一体化** | 光迅科技 | 自主可控+产业链完整 |
| **T2 追赶者** | 华工、剑桥、联特 | 弹性大，确定性略弱 |

### 4.2 CPO 过渡期的核心矛盾

- **传统可插拔业务**仍是业绩主力（800G 放量 + 1.6T 起量）
- **CPO 业务**尚未规模化，但决定长期竞争力
- **关键能力**：光引擎自研能力、硅光平台积累、客户 CPO 合作深度

### 4.3 谁会成为 CPO 时代的赢家？

**一个基本判断**：
> CPO 时代的光模块厂商，本质上要升级为"**光引擎 ODM + 系统方案商**"。拥有硅光自研 + 客户深度绑定 + 先进封装能力的厂商将胜出。

- **最优**：天孚通信（已绑英伟达光引擎）
- **次优**：中际旭创（自研硅光 + 规模优势）
- **博通生态**：新易盛
