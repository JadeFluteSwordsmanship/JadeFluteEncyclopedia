# 光芯片与光源环节（上游）

> **环节定位**：CPO 产业链的**「卡脖子」**上游。激光器芯片、CW 光源、DSP 是 CPO 核心稀缺资源。
> 
> **核心逻辑**：CPO 架构下，光源从"分散式集成在模块内"变为"集中式外置光源池（ELS）"，单台 CPO 交换机 CW 光源**用量激增 + 大功率要求**，国产光芯片迎来从 0 到 1 的增量机遇。

---

## 一、光芯片基础知识

### 1.1 光芯片分类

| 类别 | 细分 | 主要用途 | 代表厂商（海外） | 国产代表 |
|---|---|---|---|---|
| **有源芯片** | DFB 激光器 | 光模块光源 | Lumentum、Finisar | 源杰、仕佳、光迅 |
| | EML（电吸收调制） | 100G/200G 长距 | Lumentum、Mitsubishi | 源杰、光迅、长光华芯 |
| | VCSEL（垂直腔面发射） | 多模短距 | II-VI、Broadcom | 长光华芯 |
| | CW-DFB（连续波） | **CPO 外置光源** | Lumentum、Furukawa | **源杰、仕佳** |
| **无源芯片** | AWG 阵列波导光栅 | 波分复用 | NEL、Enablence | 仕佳、博创 |
| | PLC 芯片 | 分路器、干涉仪 | NEL | 仕佳、光迅 |
| | 硅光芯片 | CPO 光引擎 | Intel、Cisco | 光迅、中际旭创（自研）|

### 1.2 为什么 CPO 对光芯片是"从 0 到 1"的增量机遇？

**传统可插拔**：每个光模块内置 1-2 颗 DFB/EML，分布式

**CPO 架构**：
- 单台 51.2T CPO 交换机可能集中使用 **128-256 颗** CW 光源
- 单颗 CW 输出功率要求 **70mW+**（传统模块 10mW 即可）
- 多颗光源通过**光源池（ELS Pool）**集中外置

📌 **结论**：光源用量 × 单颗价值 → **量价齐升**

## 二、CW 光源核心标的

### 1. 源杰科技（688498.SH）—— 国产激光器芯片 IDM 龙头

| 项目 | 详情 |
|---|---|
| **公司定位** | 国内少数具备全流程 IDM 能力的高速激光器芯片公司 |
| **产品矩阵** | 2.5G/10G/25G/50G DFB、EML 芯片，**CW-DFB 硅光光源芯片** |
| **CPO 核心价值** | **25G/50G CPO 光源**已量产，良率超 **80%**，直接供货头部模块厂 |
| **核心客户** | 中际旭创、新易盛、光迅科技等头部光模块厂 |
| **技术壁垒** | MOCVD 外延 + 流片 + 封测全流程自主，国内最接近国际水平 |
| **CPO 弹性** | CW 大功率光源从 0 到 1，单颗价值量是传统 DFB 的 2-3 倍 |

**投资逻辑**：
- ✅ **CPO 最纯正的上游标的**
- ✅ 国产替代率当前 <30%，空间巨大
- ✅ IDM 模式，毛利率高
- ⚠️ 规模较小，业绩弹性大但波动性高

### 2. 仕佳光子（688313.SH）—— 光芯片+器件双主业

| 项目 | 详情 |
|---|---|

**投资逻辑**：

- ✅ FAU 业务受益于光引擎耦合需求爆发
- ⚠️ 多业务并行，聚焦度不如源杰

## 三、国产硅光与薄膜铌酸锂

### 薄膜铌酸锂（TFLN）—— 下一代调制器材料

| 特点 | 描述 |
|---|---|
| **优势** | 带宽高（>100GHz）、功耗低、CD 参数优 |
| **应用** | 1.6T/3.2T 及以上超高速调制器 |
| **国产标的** | **光迅科技**（领先）、**光库科技**（器件）、富信科技 |

### 硅光芯片（Silicon Photonics）
- CPO 光引擎的主流技术路径
- 国产厂商：光迅、中际旭创（自研）、上海微技术工业研究院
- 海外：Intel、Cisco、Ayar Labs

## 四、光通信 DSP 芯片

### 背景
光模块 DSP 是 PAM4 信号调制/恢复核心：
- **海外垄断**：Marvell、Broadcom、MaxLinear
- **国产突破**：**澜起科技（688008.SH）**、**国民技术**、**裕太微（688515.SH）** 等在研

### LPO 与 DSP-Free 趋势
- LPO（Linear Pluggable Optics）去掉 DSP，成本/功耗降低
- 对 DSP 市场形成分流
- 但 CPO 更依赖自研/集成 DSP
