# CPO 板块全景研究（Co-Packaged Optics）

> **整理日期**：2026-04-17  
> **板块定位**：AI 算力基础设施 → 通信/光互联 → CPO（共封装光学）  
> **核心判断**：CPO 是 AI 算力突破「功耗-带宽-密度」三重瓶颈的必由之路，2025 年「易中天」三剑客业绩炸裂已验证光互联高景气，2026 年进入 CPO 规模化商用元年。

---

## 一、什么是 CPO？—— 从「板级互连」走向「封装级互连」

### 1.1 基本定义

**CPO（Co-Packaged Optics，共封装光学）**：将光引擎（Optical Engine）直接与交换芯片/ASIC/GPU 封装在同一基板上，用光互连替代传统的电互连（SerDes + PCB 铜线），把电信号传输路径从 **厘米级缩短至毫米级**。

一句话概括：**不是造更强的芯片，而是重构芯片与网络的连接方式。**

### 1.2 与传统方案对比

| 对比项 | 传统可插拔光模块 | CPO |
|---|---|---|
| 架构 | 交换芯片 → PCB → 光模块 → 光纤 | 交换芯片 + 光引擎（共封装） → 光纤 |
| 电路径长度 | 厘米级 | 毫米级 |
| 功耗 | 基准 | **下降 30%~50%** |
| 带宽密度 | 1.6T 以上物理瓶颈 | 提升数倍，可支撑 3.2T/6.4T |
| 面板 I/O 密度 | 触顶 | 大幅提升 |

### 1.3 技术演进节奏

| 阶段 | 时间 | 主导方案 | 落地场景 |
|---|---|---|---|
| **Stage 1：可插拔光模块** | 已成熟 | 400G/800G 可插拔（EML/硅光） | 全场景 |
| **Stage 2：LPO / NPO 过渡** | 2025-2026 | 线性直驱 / 近封装 | 特定高速场景 |
| **Stage 3：CPO 起量** | **2026-2028** | 硅光 CPO + 外置光源 ELS | AI 集群机架间/集群级 |
| **Stage 4：CPO 主导** | 2028+ | 全光互连 | AI 数据中心全面渗透 |

> **关键触发条件**：当 AI 集群规模达到 **10 万 GPU 级别**，CPO 从"可选项"变成"刚需"。英伟达已在 GTC 2025 发布首款 CPO 交换机 Quantum-X / Spectrum-X。

### 1.4 三大工程挑战（风险点）

1. **热管理**：光引擎靠近高功耗 ASIC，散热设计难度跃升
2. **封装良率**：光电共封装制造良率是规模化最大障碍
3. **激光器寿命**：高温环境中的激光器可靠性待验证

## 二、CPO 产业链全景图

### 2.1 四层架构

```
┌──────────────────────────────────────────────────────────┐
│  下游（应用）：AI 数据中心 / 超算中心 / 云计算网络        │
│               Nvidia / Broadcom / Google / AWS / Meta    │
├──────────────────────────────────────────────────────────┤
│  中游（模块+引擎+无源器件）                               │
│  • 光模块厂：中际旭创、新易盛、华工科技、剑桥科技         │
│  • 光引擎：天孚通信（英伟达 CPO 主供）                    │
│  • 无源器件：太辰光、致尚科技、光库科技                   │
├──────────────────────────────────────────────────────────┤
│  核心环节（先进封装·CPO 成败关键）                        │
│  • 2.5D/3D 封装、硅光集成、光电协同设计                   │
│  • 台积电 COUPE、Intel Co-EMIB、JCET 封测                 │
├──────────────────────────────────────────────────────────┤
│  上游（材料+光芯片+光源+精密光学）                        │
│  • 光芯片：源杰科技、光迅科技、仕佳光子、长光华芯         │
│  • 外置光源 ELS：源杰、仕佳（CW-DFB 激光器）              │
│  • 精密光学：炬光科技、腾景科技                           │
│  • 设备：罗博特科（ficonTEC 晶圆级耦合检测）              │
└──────────────────────────────────────────────────────────┘
```

### 2.2 价值链重构逻辑

传统可插拔方案中，光模块厂商掌握 60%+ 的价值。CPO 架构下：

- 👉 **光模块厂商价值被摊薄**（封装环节向上游转移）
- 👉 **光引擎 / 光芯片 / 先进封装价值大幅提升**
- 👉 **CW 光源、FAU 等高密度无源器件价值凸显**

**一句话投资逻辑**：**短期看光模块（业绩兑现），中期看器件（结构性放量），长期看芯片+封装（价值重塑）。**

## 四、投资逻辑与关键节奏

### 4.1 确定性排序（基于 2026-04 时点）

```
天孚通信（CPO 纯受益弹性最大）
  > 中际旭创、新易盛（业绩兑现叠加 CPO 过渡）
    > 源杰科技、仕佳光子（上游光源量价齐升）
      > 罗博特科（设备先行）
        > 太辰光、致尚科技（高密度无源爆发）
          > 光迅科技（全产业链+国资+自主可控）
            > 炬光科技（稀缺精密光学）
```

### 4.2 关键催化时点

| 时间 | 事件 |
|---|---|
| 2026 H1 | 英伟达 Quantum-X / Spectrum-X CPO 交换机小批量出货 |
| 2026 H2 | 博通、Marvell CPO 方案跟进；中际旭创 3.2T 产品客户测试 |
| 2027 | CPO 大规模商用放量；硅光/薄膜铌酸锂渗透加速 |
| 2028+ | CPO 成主流，10 万卡集群标配 |

### 4.3 核心风险

- **技术路线竞争**：LPO（线性直驱）对 CPO 的分流可能
- **良率爬坡**：硅光 CPO 封装良率能否快速突破
- **估值泡沫**：CPO 标的 2026-04 PE 普遍 70 倍+，业绩兑现能否消化
- **供应链瓶颈**：EML / DSP / CW 光源高端芯片紧张
- **800G 价格战**：模块端 2026 年或承压
