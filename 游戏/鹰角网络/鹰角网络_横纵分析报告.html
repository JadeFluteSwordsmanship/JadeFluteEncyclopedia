一、一句话定义

鹰角网络是一家由《明日方舟》崛起的中国原创内容型游戏公司。它并不靠最激进的商业化、也不靠最大众的玩法取胜，而是靠“世界观 + 策略玩法 + 统一审美 + 高浓度社区认同”建立起强品牌。它更像一家世界观驱动的内容公司，而不只是传统意义上的手游厂商。

二、纵向分析：鹰角是怎么从一个小团队，长成一家具备品牌气质的公司的？

2.1 起点：它为什么不是“又一家二次元手游公司”？

如果只看外表，鹰角网络很容易被放进一个熟悉的分类里：

  • 中国公司
  • 做二次元手游
  • 靠抽卡和角色赚钱
  • 代表作是《明日方舟》

但如果你真的把它拆开来看，会发现鹰角从一开始就和很多“为了做一款能爆的手游”而成立的团队不太一样。

它成立于2017年。这个时间点很关键。

当时中国二次元游戏市场已经不是荒地了。《阴阳师》证明了二次元审美和卡牌抽卡可以做大，《崩坏3》证明了国产二次元动作游戏有技术突破的空间，FGO 在国内的影响力则让所有人意识到：玩家不只是为数值付费，也会为角色、文本和世界观付费。

但在那个阶段，市场主流的二次元手游仍然更偏两个方向：
- 角色消费与卡牌养成
- 动作/回合制RPG的工业化复制

而鹰角选了一个更怪、也更险的切口：塔防。

塔防在主流手游里并不算最性感的赛道。它不如动作游戏容易出圈，不如回合制方便做数值扩张，也不像开放世界那样自带声量。可恰恰因为它不是最拥挤的赛道，鹰角有机会把别人在别的品类里不愿意做得那么深的东西——世界观、文本、音乐、视觉风格、社区文化——一次性打包进去。

从公司官网的表述看，鹰角一直很强调一句话：“逻辑艺术”。

这个说法其实非常鹰角。它不是一个纯商业表达，也不是一句空洞品牌口号。它隐含的意思大概是：

  • 游戏不是单纯爽感机器
  • 也不是只讲故事的内容容器
  • 它是一种由系统规则、叙事结构、美术风格和玩家体验共同组成的作品

这决定了鹰角后来一系列很重要的选择：
- 不走最大众的玩法起手
- 不把角色消费做成唯一核心
- 不把世界观当贴皮
- 不把音乐、美术、PV当宣传附属

你可以说，它从一开始就想做的，不只是“一个手游项目”，而是“一个有世界、有气质、有自我表达的产品”。

2.2 创始团队：技术、二次元审美和内容野心的交叉点

鹰角成立于上海，法定代表人为黄一峰。公开信息里，黄一峰与制作人海猫络合物是最核心、也最常被讨论的两位人物。

如果粗暴概括，这个团队的早期气质是：
- 技术底色很强
- 二次元文化理解很深
- 对内容表达有明显野心

这三件事同时成立，才是鹰角真正稀缺的地方。

因为中国游戏行业里并不缺懂技术的人，也不缺懂二次元的人，但真正罕见的是：既有工程能力，又有世界观构建能力，还愿意长期围绕一个IP慢慢堆东西的人。

很多公司做二次元产品，最强的是角色包装能力；很多公司做大DAU产品，最强的是商业化运营能力。鹰角更像另一种形态：它先把自己当成“内容创作者”，然后再用游戏这个媒介把内容做出来。

这就解释了为什么鹰角后来的很多动作——塞壬唱片、动画、线下展、品牌服装、周边体系——看起来不像“手游公司顺手做点衍生”，而更像是“这家公司本来就想把一个世界做完整”。

2.3 《明日方舟》的诞生：为什么是塔防？为什么是这种气质？

《明日方舟》是鹰角真正完成市场定义的作品。

如果你没玩过，我先用最直白的话解释：

它是一款“塔防 + 角色养成 + 抽卡”的游戏。

你在战斗里不是操纵一个角色满地图跑，而是在地图上部署不同职业的“干员”，拦住敌人、控制路线、打输出、治疗队友。战斗不是比反应快，而更像在做策略题：
- 谁先下场
- 哪个位置卡住敌人
- 什么时候开技能
- 哪种职业配合更合理

从这一点上，它和很多主流二次元手游已经不一样了。

大多数二次元手游更偏“数值成长 + 角色魅力 + 自动或半自动战斗”，而《明日方舟》把“关卡解题感”做得很重。你不是单纯养强角色就自动赢，很多时候你得真理解机制。

更特别的是，它在策略玩法之外，又叠上了一层非常厚的世界观和剧情。

《明日方舟》的故事发生在名为“泰拉”的架空世界。这个世界的关键词包括：
- 天灾
- 源石
- 矿石病
- 感染者
- 移动城市
- 国家、阶层、战争、歧视与政治博弈

如果非要找现实类比，它有一点末世工业感，有一点政治寓言感，也有一点“架空历史小说”的味道。它不是单纯讲主角冒险升级，而是在不断讲一个世界怎样失序、怎样分裂、不同人如何在结构性压迫中挣扎。

所以对很多玩家来说，《明日方舟》不是“我抽到了哪个老婆”，而是“我进入了一个很完整、很严肃、很有质感的世界”。

这也是它和一般抽卡手游气质差异最大的地方。

2.4 如果你没玩过明日方舟，它玩起来到底像什么？

我换一种更贴近玩家体验的说法。

《明日方舟》玩起来，不像那种你一打开就立刻爽的动作手游，也不像很多“挂机 + 养成”的轻松抽卡游戏。它更像：

  • 一部分是策略塔防
  • 一部分是角色收集和培养
  • 一部分是长期追更世界观小说
  • 再加一层非常强的美术、音乐和角色氛围消费

优点很明显：
1. 策略感强：你会觉得自己在解题，而不是单纯看数值碾压
2. 角色不只是数值单位：每个干员都像世界的一部分
3. 剧情很厚：如果你爱读设定，会非常上头
4. 音乐与PV质量很高：很多人甚至是先被PV和音乐打动
5. 日常肝度不算最重：相比一些长线二游，明日方舟经常被评价为“低肝低氪”

缺点也很明显：
1. 门槛高：玩法不是最直觉，剧情也不是最轻松
2. 文本密度高：很多人会吐槽“像在读论文”“像在啃设定集”
3. 新手容易有压迫感：系统、职业、资源、剧情量同时压上来
4. 它不太迎合“快速爽感”：如果你要的是轻松刷一会儿，它可能反而显得重

所以它更像一款会筛用户的产品。它不一定适合所有人，但适合的人会很喜欢。

2.5 爆发之后：鹰角不是“做完爆款”，而是开始把爆款变成宇宙

2019年《明日方舟》上线并迅速成功之后，鹰角没有走最常见的路线——那种“趁热复制同类产品”“快速上第二款换皮爆款”的路线。

它反而做了另一件事：把《明日方舟》从一个游戏，慢慢做成一个完整的内容宇宙。

这一步非常关键。

很多公司也会做周边、联动、动画，但鹰角做这些事的时候，外界常常会觉得它不是在“变现衍生”，而是在“继续完善世界”。原因就在于，它的延展和主产品调性高度一致：

  • 音乐：塞壬唱片不是简单的OST集合，而像是独立厂牌在做世界内音乐表达
  • 动画与PV：强化世界感，而不是只做转化广告
  • 服装和线下活动：不只是卖周边，而是在做品牌生活方式延展
  • 角色、场景、美术、展会体验：共同强化“罗德岛”这个想象共同体

这就是为什么鹰角的玩家关系会和很多同行不一样。

很多二次元游戏的玩家和公司之间，关系核心是“角色消费”和“版本期待”；鹰角和玩家之间，关系核心里还额外多了一层：共同维护一个世界的感觉。

你会发现，玩家对鹰角的认同有时候不是“我最喜欢哪个角色”，而是“我喜欢鹰角做内容的方式”“我喜欢这个世界的气质”。

这使鹰角变得更像一个品牌，也更像一个作品型厂牌。

2.6 社区文化：为什么鹰角用户看起来比很多二游用户更“挑剔”？

鹰角用户是很有意思的一群人。

如果概括一句：

他们不是最纯粹的抽卡强度党，而更像一群内容型玩家。

这批玩家通常对这些东西特别敏感：
- 世界观有没有漏洞
- 文本写得好不好
- PV有没有审美
- 音乐有没有灵魂
- 角色设计是不是有辨识度
- 活动和线下展是不是“像鹰角”

这会导致一个后果：他们的忠诚度很高，但容忍度也不一定高。

也就是说，鹰角一旦做得对，用户会非常认同，甚至形成强烈的品牌归属；但一旦做得“不像自己”，用户会比很多大众产品玩家更敏感地感受到“失调”。

这也是为什么外界经常说鹰角“调性强”。

它的“调性”不是一句玄学评价，而是指：
- 世界观气质统一
- 美术语言统一
- 音乐风格统一
- 宣传物料统一
- 社区想象统一

这种统一性一旦成立，公司本身就会变成一个可被消费的品牌对象。

换句话说，玩家消费的已经不只是游戏，而是“鹰角”这家公司所代表的那种内容品位。

2.7 《终末地》：鹰角第一次真正往“大制作工业化”方向迈步

如果《明日方舟》证明了鹰角能做出一个独特且成功的作品，那么《终末地》要证明的事情就更大了：

鹰角能不能从“单一爆款 + 强品牌”成长为“多项目 + 大工业化能力”的公司？

《终末地》之所以重要，不只是因为它是一个新项目，而是因为它承载了鹰角所有最危险也最诱人的升级方向：

  • 从2D到3D
  • 从塔防策略到更大体量的即时战斗 / 探索 / 基建体验
  • 从手游型项目到全平台工业化产品
  • 从小众高粘性产品到更可能破圈的内容载体

更复杂的是，《终末地》并不是和《明日方舟》完全无关的新IP，它又共享“明日方舟”这个标签。这使它天然同时拥有两种力量：

加成：
- 老玩家信任
- 世界观认知基础
- 品牌势能

压力：
- 玩家会拿它和《明日方舟》的气质做对照
- 玩家会期待它既更大、更强，又不能失去原来的灵魂
- 一旦不及预期，反噬也会更强

所以《终末地》在商业上当然重要，但它更像是一场品牌级考试。

如果它成功，鹰角就不会只是“靠明日方舟活得很好的一家公司”，而会进一步变成真正意义上的内容型多产品公司；如果它失手，外界会怀疑的是鹰角整个从“作品型工作室”走向“工业化内容公司”的能力边界。

2.8 鹰角和同行比，究竟特别在哪？

这是最值得横着看的地方。

如果把几家中国内容型/二次元游戏公司摆在一起：

  • 米哈游更像技术驱动的全球内容工业平台
  • 叠纸更像女性向情感内容与审美消费公司
  • 库洛更像动作系统和3D战斗体验驱动公司
  • 散爆更像设定浓度极高的小众军武世界观品牌

而鹰角的位置非常微妙。

它不是技术力叙事最强的，也不是商业化最重的，也不是最破圈的大众型公司。但它可能是“世界观驱动的内容公司”这个位置上，最有辨识度的一批。

它的独特之处在于：
1. 玩法并不主流，但成功了
2. 内容气质比数值刺激更重要
3. 社区共识围绕“作品感”而不是单纯“老婆感”形成
4. 音乐、美术、PV、活动的品牌整合度很高

所以我会把鹰角放在一个很特别的位置上：

它像中国二次元游戏行业里的“作品型内容厂牌”。

不一定最大，但往往最有“作者性”气质。

2.9 上市状态：为什么我把它放进“游戏目录”，而不是“个股库”

我先说结论：目前按较可靠公开信息判断，鹰角网络未上市。

原因很简单：
- 工商主体仍是有限责任公司
- 没查到真实交易所挂牌信息
- 一些第三方网站出现的疑似代码/行情页，不足以证明真实上市，更多像自动生成页面或占位符

所以按你的规则，这份 PDF 我会放进研究目录的 游戏子目录，而不是个股库。

这件事本身其实也说明了鹰角的一种状态：它虽然已经是一家很有体量、很有品牌影响力的公司，但仍然保持相对私有化、相对独立的组织姿态。这和它整体给人的印象是一致的——没那么资本化表演，更像认真做内容的团队。

三、横向分析：鹰角到底像谁，又不像谁？

3.1 鹰角 vs 米哈游：不是同一条增长路线

很多人喜欢拿鹰角和米哈游放在一起，因为它们都代表中国二次元内容工业的头部样本。

但它们的路径差异其实很大。

米哈游的核心能力更像：
- 高规格技术与工业化
- 大体量产品组织能力
- 全球同步发行和全球市场表达
- 把角色、剧情、世界和商业化整合成更大的平台体系

鹰角则更像：
- 先用一个强气质产品建立品牌
- 再把这个品牌往更深的世界观和IP方向做厚
- 不是先求最大，而是先求最像自己

所以如果米哈游更像“工业化内容巨头”，鹰角更像“作者型内容公司”。

这也解释了为什么《终末地》的意义会被特别放大——它是鹰角第一次真正向米哈游式的大工业能力逼近。

3.2 鹰角 vs 叠纸：都重美术和品牌，但卖的不是同一种东西

叠纸擅长的是女性向情感内容和审美消费。它让玩家沉浸在角色关系、情感体验和精致生活方式里。

鹰角也重美术、重周边、重线下品牌，但它卖的不是“恋爱想象”或“生活方式梦境”，而是：
- 世界观厚度
- 末世工业审美
- 策略与叙事的严肃感
- 对一个架空社会的认同感

所以两家公司都像内容型品牌，但：
- 叠纸更像卖情感氛围
- 鹰角更像卖世界气质

3.3 鹰角 vs 库洛：系统手感和世界厚度的分野

库洛这几年最鲜明的标签是动作体验和3D内容工业化能力。

而鹰角最鲜明的标签一直不是手感，而是设定、叙事、审美统一性和品牌气质。

所以《终末地》如果要往更大众的3D方向走，它一定会被拿来和库洛、米哈游比较。但这并不意味着它必须复制它们的成功公式。

终末地最有希望成立的路径，恰恰不是“把自己做成另一个原神或鸣潮”，而是：

把鹰角原本那套世界观与气质优势，迁移到更大的3D工业化壳子里。

这条路更难，但一旦成功，就不再只是跟随者。

3.4 鹰角 vs 散爆：同样重设定，但鹰角更完整地把“内容”做成了品牌

散爆和鹰角其实有一些相似性：
- 都有很强的设定浓度
- 都吸引愿意深挖世界观的圈层用户
- 都不是纯靠大DAU玩法起飞

但鹰角更进一步的地方是，它把内容表达整合成了更完整的品牌体系：
- 音乐厂牌化
- 活动与展会更成熟
- 视觉语言更统一
- 对外品牌形象更清晰

所以散爆更像深设定的小众品牌，鹰角则更像把深设定品牌进一步工业化、品牌化了。

四、横纵交汇洞察：鹰角真正稀缺的，不是《明日方舟》这一个产品，而是“作品感公司”的形成

4.1 鹰角为什么总被认为“调性很强”？

因为它的很多系统不是并列存在，而是互相强化的：
- 世界观强化角色
- 角色强化剧情
- 剧情强化音乐
- 音乐强化PV
- PV强化线下活动
- 活动又反过来强化玩家对整个品牌的归属

也就是说，“调性”不是做出来的一张海报，而是一整套内容系统彼此一致之后产生的结果。

4.2 《明日方舟》为什么气质和一般抽卡手游不同？

因为它的核心吸引力不是“角色有多可爱”“强度有多夸张”，而是：
- 玩法需要脑子
- 世界观需要阅读
- 文本需要耐心
- 社区需要共鸣

它当然也是抽卡游戏，但抽卡在这里更像维持长线经营的商业机制，而不是整个产品精神的全部。

4.3 鹰角最大的优势和最大风险是什么？

最大优势：
- 高粘性社区
- 强品牌认同
- 世界观和审美统一性极强
- IP外延能力好
- 玩家不是只消费角色，而是在消费公司所代表的气质

最大风险：
- 成功高度系于少数核心IP
- 用户对“失调”容忍度低
- 从2D强品牌公司升级为3D工业化公司难度极高
- 一旦《终末地》无法兑现用户预期，反噬会比普通项目更大

4.4 未来推演：终末地成功之后，鹰角会变成什么？

我觉得它不会简单变成“另一个米哈游”。

更可能的情况是：

如果终末地成功，鹰角会成为一种更少见的公司：既保持强烈作者性和世界观调性，又拥有较强工业化制作能力的多产品内容公司。

这意味着它会从“单一爆款 + 强IP”迈向“多产品 + 强品牌 + 强内容工业”。

如果做成，它在中国游戏行业里的位置会非常特别。

五、结论：鹰角像什么？

研究到最后，我觉得最贴切的说法不是“它是一家很会做二次元手游的公司”，而是：

鹰角是一家先把自己当成内容创作者，再把游戏当成载体的公司。

它不是最大众的，不是最商业化激进的，也不是最技术炫耀型的。

但它可能是中国二次元游戏行业里，最像“作品型厂牌”的一批公司之一。

《明日方舟》的成功不是一个偶然爆款，而是这家公司世界观能力、审美统一能力、策略设计能力和社区文化能力共同堆出来的结果。

而《终末地》真正重要的，也不是它会不会再赚一笔大的，而是它能不能证明：

鹰角不是只能做一款强作品，而是有能力把“作品感”扩展成一个更大的公司级能力。

如果这一步走成，鹰角就不只是一个二次元手游厂商，而会更像一家真正的内容公司。
