# MEMORY.md - 长期记忆

## 项目背景

这是一个**投资百科全书**式的 Knowledge Base，组织结构按"行业 → 产业链分层 → 子环节 → 标的"。

## 项目约定

### 目录组织
- `AI全行业/` — AI 基础设施/模型/应用三层
  - `01_Infrastructure_Hardware/` — 硬件（算力/存储/光通信）
    - `1_Compute_Chips/` `2_Memory_Chips/` `3_Communication_Optical/`
    - `3_Communication_Optical/` 下分 `3.1_Optical_Module/` 和 `3.2_CPO/`
- `人形机器人/` — 上游零部件/中游 OEM/下游应用/具身智能/深度
- `个股库/` — 个股深度研究档案

### 文件命名规范
- 个股文件：`<代码><公司名>.md`，如 `300308中际旭创.md`、`300475香农芯创.md`
- 板块总览：`README.md`
- 主题分拆：`0X_主题名.md`（带数字前缀排序）

### 个股 md 内容规范
已有格式参考 `300475香农芯创.md`、`301308江波龙.md`：
- 顶部可带研报摘录（时间、机构、评级）
- 或采用结构化分析（板块标签、业绩、看点、风险、跟踪清单）
